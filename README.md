# AnkiQuickCard

## 卡片生成Prompt

以下是专门用于生成初步Anki卡片文本的prompt，适用于法律法规/工程标准等内容的卡片制作：

```
# Role
你是一位深耕【法律法规/工程标准】的 Anki 笔记架构师，擅长将《注安》、《一消》等教材条文转化为严格符合“原子化记忆”原则的 Anki 卡片。

# Task
将我提供的文本拆解为可直接导入 Anki 的混合型卡片。

# 核心规则 (严格执行)
1. **原子化拆分**：一张卡片仅限一个考点。禁止长段落，确保手机一屏显示，不滚动。
2. **主语补全**：严禁使用代词（它/该/其）。必须在每张卡片开头用<b></b>加粗主体。
   - *错误*：其应当在30日内上报。
   - *正确*：<b>建设单位</b>应当在{{c1::30}}日内上报。
3. **挖空策略 (考试导向)**：
   - 仅针对：数字、天数、金额、资质等级、责任主体、核心关键词。
   - 每张卡片 1~3 个空。同一逻辑的空用 {{c1::}}，递进逻辑用 {{c1::}} {{c2::}}。
4. **题型自动选择**：
   - 描述性条文使用 **Cloze (填空)** 格式。
   - 纯定义或独立问题使用 **Basic (问答)** 格式。

# 导出格式 (适配 Python 脚本)
- 每张卡片之间必须使用独占一行的 `---` 分隔。
- **Cloze 格式**：直接输出带 {{c1::}} 的单行文本。
- **Basic 格式**：第一行为问题，第二行为答案。
- **禁止输出**：序号、标题、Markdown 格式化（加粗/斜体）、解释说明。

# 示例参考
<b>生产经营单位</b>的主要负责人未履行安全生产管理职责的，责令{{c1::限期改正}}；逾期未改正的，责令{{c2::停产停业整顿}}。
---
<b>事故调查组</b>应当自事故发生之日起{{c1::60}}日内提交事故调查报告。
---
什么是“三同时”制度？
建设项目的安全设施，必须与主体工程<b>同时设计、同时施工、同时投入生产和使用</b>。

# Input
请发送需要处理的考点文本，我将直接输出成品卡片。
```

## 项目介绍

Anki Connect卡片推送工具，用于快速将文本转换为Anki卡片并推送到Anki软件中。

## 功能特点

- 支持解析文本生成两种类型的Anki卡片：
  - 基础卡（正面/背面）
  - Cloze卡（填空题格式）
- 自动检测卡片类型
- 支持自定义标签和牌组
- 实时预览转换结果
- 批量推送卡片到Anki
- 连接测试功能
- 简单直观的图形界面

## 安装说明

### 1. 安装Anki和Anki Connect

- 首先需要安装[Anki](https://apps.ankiweb.net/)
- 在Anki中安装[Anki Connect](https://ankiweb.net/shared/info/2055492159)插件
  - 打开Anki → 工具 → 插件 → 获取插件 → 输入代码 `2055492159` → 安装 → 重启Anki

### 2. 安装Python依赖

```bash
pip install -r requirements.txt
```

### 3. 运行程序

```bash
python main.py
```

## 使用方法

1. **启动Anki**：确保Anki已启动并加载了Anki Connect插件
2. **测试连接**：点击"测试连接"按钮确认与Anki Connect的连接状态
3. **输入文本**：在左侧"原始文本"区域输入要转换的文本
4. **解析文本**：点击"解析文本"按钮，系统会自动解析文本并在右侧"转换预览"区域显示结果
5. **配置选项**：
   - 输入标签（必填）
   - 选择或输入牌组名称（必填）
6. **推送卡片**：点击"推送至Anki"按钮，系统会将解析后的卡片推送到Anki中
7. **查看结果**：推送完成后会显示推送结果，包括成功和失败的卡片数量

## 文本格式说明

### 基础卡格式

使用`---`分隔不同卡片，每张卡片的第一行为正面，其余行为背面：

```
卡片1正面
卡片1背面
---
卡片2正面
卡片2背面第一行
卡片2背面第二行
```

### Cloze卡格式

使用`---`分隔不同卡片，使用`{{c1::}}`、`{{c2::}}`等格式标记填空内容：

```
{{c1::Python}}是一种{{c2::解释型}}编程语言
---
{{c1::Anki}}是一个{{c2::记忆}}卡片软件
```

### 混合格式

可以在同一文本中混合使用基础卡和Cloze卡格式：

```
Python
一种解释型编程语言
---
{{c1::Anki}}是一个{{c2::记忆}}卡片软件
```

## 注意事项

1. 必须先启动Anki并加载Anki Connect插件
2. 标签不能为空，且会自动过滤非法字符
3. 牌组名称不能为空，若不存在会自动创建
4. 推送过程中请不要关闭Anki
5. 为避免Anki卡顿，每推送10张卡片会暂停0.5秒
6. 推送完成后会显示详细的推送结果

## 技术实现

- 使用Python的tkinter库构建图形界面
- 使用requests库与Anki Connect API进行通信
- 使用正则表达式解析文本格式
- 使用多线程避免UI阻塞


## 常见问题

### Q: 连接失败怎么办？
A: 请检查：
   - Anki是否已启动
   - Anki Connect插件是否已安装并启用
   - 网络连接是否正常

### Q: 推送失败怎么办？
A: 请检查：
   - Anki是否仍在运行
   - 牌组名称是否正确
   - 卡片内容是否符合Anki的格式要求

### Q: 如何创建新牌组？
A: 在牌组输入框中直接输入新的牌组名称，系统会自动创建
